### 任务：
* 完成多道批处理作业调度。

### 要求：
1. 只考虑1个CPU的资源，其他资源不考虑。
2. 使用**响应比高者优先算法**。

### 输入输出：
#### 输入：
```
    k               作业数(k>0)
    TJ1 YS1         提交时间 运行时间（xx时yy分）
    ...
    TJk YSk
```
#### 输出：
* 按照作业调度的先后次序输出结果。
* 输出格式： 
```
    调度次序 作业号 调度时间 周转时间 带权周转时间
    ...
    平均周转时间 平均带权周转时间
```
